# Rect2D データモデル

## 構造体: Rect2D

`Rect2D`は、2D空間内の長方形を表します。

### フィールド

| フィールド名 | 型             | 説明                   |
|--------------|----------------|------------------------|
| `Position`   | `gummymath.Vector2D` | 長方形の左上の角の座標。 |
| `Width`      | `float64`      | 長方形の幅。           |
| `Height`     | `float64`      | 長方形の高さ。         |

### 検証ルール

- `Width`と`Height`は非負の値でなければなりません。
